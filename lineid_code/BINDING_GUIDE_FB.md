## 🔔 什麼是「綁定」？為什麼要綁？

「綁定」就是把您的手機 LINE 和您在**賜安診所的病歷號**連結起來。
綁定之後您可以從 LINE 直接查詢以下個人就醫資訊：

| 功能 | 說明 |
|------|------|
| 📋 **欠單查詢** | 查看押單紀錄 |
| 💉 **抽血日期查詢** | 知道什麼時候抽過血 |
| 💊 **慢性病領藥查詢** | 看慢連箋第幾次領藥、什麼時候該領 |

**不綁定也能用**：門診表、醫師介紹、AI 客服、看診時段⋯⋯這些公開資訊不用綁定就能查。

## ✅ 綁定流程（4 步驟搞定）

### Step 1 綁定人和被綁定人親自持健保卡和身分證到診所櫃檯，向服務人員說
> 「我要綁定 LINE，查詢就醫資料」，櫃台人員會核對身份

🔎 小提醒：如果是家人代替操作 LINE，請同時說明「我是幫家人綁定」。

## 🔓 如何解除綁定（取消連結）

日後不想用了，或想換別的 LINE 帳號綁定，隨時都可以解除！

### 解綁步驟：

1. 開啟 LINE「**賜安診所**」聊天室
2. 點選下方「**選單**」
3. 點「**🔐 查詢就醫資訊**」
4. 系統會列出您綁定的病歷號，點「**選擇**」
   病歷號採遮罩格式，LINE Bot 顯示給用戶時，recno 需去識別化：
  - 格式：`第一碼*****最後一碼`
  - 範例：`036787` → `0*****7`
5. 點「**❌ 取消綁定**」
6. 系統會再問您一次「確定要取消嗎？」
7. 點「**是，解除綁定**」✅ 完成！

LINE 會回覆您：

> ✅ 已解除綁定。未來如需使用 LINE 查詢功能，請至櫃台重新綁定。

## ❓ 常見問題 Q&A

### Q1️：驗證碼過期怎麼辦？
A：到櫃台重新跟服務人員說「我要再申請一組驗證碼」就好囉！
驗證碼有效時間是 **5 分鐘**，過期就失效，需重新申請。

### Q2️：可以幫家人綁定嗎？
A：可以！例如爸爸幫媽媽操作 LINE，也可以綁定媽媽的病歷號。
請到櫃台時告知「**我是幫家人綁定**」，服務人員會引導您。

### Q3️：一個 LINE 帳號可以綁定多人嗎？
A：可以！例如子女的 LINE 帳號可以同時綁定爸爸、媽媽、自己的病歷號，之後選單可以切換查看。

### Q4️：解綁以後，之前查得到的資料還會留下來嗎？
A：解除綁定只是斷開 LINE 與診所的連結，您的**就醫紀錄不會刪除**，下次重新綁定仍可查詢。

### Q5️：我的手機壞了/換 LINE 帳號怎麼辦？
A：請到診所櫃台，告知「我的手機換了，需要重新綁定」，服務人員會協助您處理。

### Q6️：輸入驗證碼後沒收到回應？
- 請確認 LINE 對話對象是**「賜安診所」**官方帳號，不是 LINE 客服或其他人
- 確認輸入的是** 6 位純數字**（沒有英文字母、符號、空格）
- 確認還在 5 分鐘有效期內


### Q7️：我怕個資外洩，安全嗎？
A：請您放心～
- 您的 LINE ID、病歷號在系統中都是**加密儲存**
- 別人就算知道您的 LINE，也查不到您的病歷號
- 只有您親自輸入驗證碼才能綁定
- 隨時都可以解除綁定
